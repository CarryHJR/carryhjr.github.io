vnc server and client install and config

vnc server

windows

https://www.realvnc.com/en/connect/download/vnc/
下载最新版
安装完会提示注册用户激活，选择register offline，然后挑个激活码填进去

设置密码

右上角点击options，选择vnc password
结束，可以在privacy里面限制一些客户端的行为

xfce desktop

install

then vim ~/.vnc/xstartup

change default monitor port

把里面的5900全改成想要的端口，访问的时候记得加上用户数量，比如单用户就是5901

restart

vnc+ssh
