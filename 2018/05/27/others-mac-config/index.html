常用快捷键

command + w 关闭当前窗口
全选是option+a

shift+command+. 显示隐藏文件

mac山的mission control:触摸板四指向上或者control+up

  1. 可以在mission control 里增加desktop
  2. 显示当前desktop里的app，方便switch(app可以在menu里分配desktop)

软件全屏时会自动分配一个新的desktop
cmd+i 查看属性
cmd+space spotlight
cmd+up 到顶部
cmd+down 到尾部
cmd+tab 切换app
cmd+delete 删除
cmd+shift+delete 清空回收站
cmd+~ 同一个app中切换窗格
cmd+w 关闭当前标签
space 预览
